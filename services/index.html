Welcome to Hydroserver WaterOneFlow Web Service

What is a web service?

The Web service is be used by other websites and computer programs to access the scientific observations and measurements from this Hydroserver website in the standard WaterML format. These programs are: HydroDesktop, HydroExcel and register it in HIS Central catalogue.

By providing the web service, this website becomes a part of the worldwide CUAHSI Hydrologic Information System network for sharing scientific data with professional and citizen scientists to achieve common goals.

What are citizen scientists? They can be anyone who collects and shares scientific data with professional scientists to achieve common goals.
